# springboot-with-react

## prerequirements

- java 1.8.x
- node 12.x
- npm 6.x


## how to run

```
npm install
npm run build
./gradlew bootRun
```


## Java

- build WAS
  ```
  ./gradlew build
  ```

- run WAS
  ```
  ./gradlew bootRun
  ```


## JSX

- install dependencies
  ```
  npm install
  ```

- build
  ```
  npm run build
  npm run build:watch
  ```
